# in-Sanitty-
Just a software representation of the insane world of work

# compile:
g++ -std=c++17 -O2 -o sanity_drop main.cpp
./sanity_drop

# what it does

- Auto-start the 5-day simulation
- Randomly pick lines for each character every round
- Animate the sanity bar dropping after each full exchange
- Keep everyone perfectly in character while the smart one slowly unravels.
